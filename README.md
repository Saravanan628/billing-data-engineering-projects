# Billing Data Engineering Projects

This repository contains two real-world backend/data engineering projects simulating customer onboarding and billing workflows in SaaS systems.

## 🔹 Project 1: Customer Data Migration Pipeline

* Cleaned and validated multi-source billing data (customers, subscriptions, usage)
* Merged datasets and applied business logic
* Migrated data via REST APIs
* Implemented logging and summary reporting

## 🔹 Project 2: Usage-Based Billing Engine

* Processed usage data and validated records
* Grouped data by customer and computed billing
* Generated structured billing JSON with breakdown
* Integrated API sync, logging, and reporting

## 🛠️ Tech Stack

* Python
* REST APIs
* CSV / JSON processing
* Logging & automation scripts

## 🎯 Key Skills Demonstrated

* Data cleaning & validation
* API integration
* Data transformation & aggregation
* Debugging and error handling
* Automation scripting
