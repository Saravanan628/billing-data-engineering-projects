# Customer Data Migration Pipeline

## 📌 Overview

Simulates onboarding of new customers by migrating billing data from multiple sources into a system.

## 📁 Input Data

* customers.csv
* subscriptions.csv
* usage.csv

## ⚙️ Features

* Data cleaning and validation
* Multi-source data merging
* Billing calculation logic
* API integration using POST requests
* Logging success and failure cases
* Summary reporting

## ▶️ How to Run

```bash
python onboarding_prog.py
```

## 📊 Output

* Processed data sent to API
* migration_log.txt generated
* Summary printed in console

## 💡 Learning Outcome

Built a real-world data migration pipeline similar to SaaS onboarding workflows.
