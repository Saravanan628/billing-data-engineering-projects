# Usage-Based Billing Engine

## 📌 Overview

Simulates billing calculation based on customer usage data.

## 📁 Input Data

* usage_data.csv

## ⚙️ Features

* Data validation and cleaning
* Grouping usage by customer
* Billing calculation (usage × price)
* JSON transformation with breakdown
* API integration
* Logging and reporting

## ▶️ How to Run

```bash
python main.py
```

## 📊 Output

* Billing data sent to API
* billing_log.txt generated
* Summary of revenue and customers

## 💡 Learning Outcome

Implemented a usage-based billing system similar to real SaaS platforms.
